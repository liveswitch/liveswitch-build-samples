LiveSwitch Build SDK Web Samples

Explore a list of simple web applications showcasing various use cases using the LiveSwitch Build SDK.

Samples

  • Muting devices where indicator light turns off

    There are several ways to mute media in the LiveSwitch SDK. This sample shows how to mute and have the indicator light turn off.

  • Setting a bitrate on the upstream

    This allows you to impose a bitrate on the upstream. The quality change in the remote video can be observed.

  • Setting a bitrate on the downstream

    This allows you to impose a bitrate on the downstream. This will result in the sender adjusting the bitrate which they send.

  • Join with media muted

    When joining a session a user's devices won't be muted by defaut. This sample shows how to join a session with the microphone and camera already muted.

  • Selecting devices before joining

    Device selection usually happens automatically. This sample shows how to allow for selecting a device before joining a session.
